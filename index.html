☁️ ตั้งค่า Cloud Sync ข้ามอุปกรณ์
เชื่อมต่อ Firebase เพื่อให้ข้อมูลเหมือนกันทุกเครื่อง
วิธีตั้งค่า (ฟรี 100%) — ใช้เวลา 3 นาที
1.ไปที่ console.firebase.google.com → สร้างโปรเจกต์ใหม่ฟรี (ชื่ออะไรก็ได้)
2.เข้า Project Settings → Your apps → Web app → ลงทะเบียนแอป → คัดลอก firebaseConfig
3.เข้า Firestore Database → Create database → Start in test mode
4.เข้า Authentication → Get started → Anonymous → Enable
5.วาง firebaseConfig JSON ด้านล่าง แล้วกด เชื่อมต่อ
✅ Cloud Sync เชื่อมต่อแล้ว — ข้อมูล sync ข้ามอุปกรณ์ทุกเครื่องแบบเรียลไทม์
